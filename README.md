# FileFinch

Reliable file format detection library written in Rust.

FileFinch can identify various data formats by examining file contents and magic bytes.

## Features

- Detects multiple file formats including:
  - Apache Arrow
  - Apache Parquet
  - CSV
  - Excel (XLS/XLSX)
  - GeoJSON
  - GeoPackage
  - Shapefile
  - More to come
